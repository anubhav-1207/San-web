The San Programming Language

A clean, lightweight, and logically structured interpreted language designed for simplicity and strict evaluation.

Download v1.0 Read the Docs

🐢 Expressive Syntax

San removes unnecessary boilerplate, allowing you to write highly readable algorithms and data structures with minimal cognitive load.

💥 Robust Interpreter

Built from the ground up with a dedicated lexer, recursive descent parser, and a strictly evaluated AST for predictable execution.

✍️ Built for Learning

An excellent tool for understanding compiler design, semantic analysis, and how high-level instructions translate to machine logic.

# Recursive Fibonacci implementation in San

func fibonacci(n) {
    if n <= 1 {
        return n;
    }
    return fibonacci(n - 1) + fibonacci(n - 2);
}

let result = fibonacci(10);
print("The 10th Fibonacci number is: ", result);
